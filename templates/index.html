What is HealthCAD?

An idea thought can only be called a solution if it solves a potential problem.

This is HealthCAD, a software cum hardware system that would enable the citizens to know the accurate results about the problem they face, so that they can receive proper treatment beforehand. Our product also accumulates the data which is uploaded by user for getting predictions which helps in tuning our model better and improve with time. It can also act as a second opinion to doctor for verification purposes as well. The device can diagnose various diseases which range from heart diseases to pneumonia.
