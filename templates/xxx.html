Pneumonia Detection

This is a image classifier based on chest xray based pneumonia classification dataset. The dataset consists of two classes pneumonia and normal.

How to use the classifier:

1. Click the choose button, this will give you a pop up for selecting the file.
2. Now select the image file you want to classify(Image should be of one of the above mentioned class or the classifier won't work.).
3. Once the image is uploaded you will be able to see it on the screen, now press predict button.
4. Once the prediction is done, image will appear on screen, this image consists of two parts, the confidence score of top 5 performing class and the class activation map.

What is class activation map:

Class activation map(cam) is simple idea to visualize where in image our classifier is looking. It can be helpful for the cases where we want to observe any anomaly in image, for example in case of detecting pneumonia in chest xray, the cam will tell where it was looking at the time of classifying the image and this can help doctors visualize the results and rectify problems better.
